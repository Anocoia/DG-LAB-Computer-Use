# DG-LAB 电脑互动控制器

根据电脑使用行为实时控制 DG-LAB (郊狼) 设备输出强度的桌面应用。

通过监测 CPU 负载、鼠标键盘操作、前台应用切换等事件，经由 7 个可独立配置的模块计算输出强度，再通过 WebSocket 发送到 DG-LAB 设备。

## 功能一览

### 监控源

| 监控器 | 采集内容 |
|--------|----------|
| 系统监控 | CPU、内存、网络 I/O、磁盘 I/O |
| 输入监控 | 鼠标点击/移动/滚轮、键盘按键 |
| 应用监控 | 当前前台窗口标题与进程名 |

### 功能模块

| 模块 | 说明 |
|------|------|
| 系统监控 | 将 CPU/内存/网络/磁盘指标映射为输出强度，支持逐项配置曲线和权重 |
| 输入事件 | 鼠标点击、移动、滚轮和键盘按键转化为强度脉冲，支持连击检测和单独按键规则 |
| 定时触发 | 按固定或随机间隔周期性触发强度脉冲，可配置持续时间和随机波形 |
| 闲置惩罚 | 用户无操作时逐渐加强输出，恢复操作后立即或渐进归零 |
| 应用聚焦 | 根据前台应用匹配规则（关键字/正则）输出强度，支持黑白名单模式和切换脉冲 |
| 打字节奏 | 根据打字速度和节奏同步输出强度，支持特殊按键加成和持续打字检测 |
| 骰子惩罚 | 掷骰子随机抽取惩罚项，可配置惩罚池、轮盘模式和冷却时间 |

### 其他特性

- **双通道独立控制** — 每个模块可分别指定输出到 A/B/AB 通道
- **多模块混合** — 支持最大值、叠加、平均三种混合模式
- **全局参数** — 通道上限 (0-200)、全局倍率 (0-5x)、平滑系数
- **波形编辑** — 8 种预设波形（呼吸、脉冲、连续、锯齿、正弦、随机、心跳、潮汐），支持自定义关键帧编辑和回放控制
- **置顶小窗** — 始终置顶的迷你监视器，实时显示 A/B 通道强度和各模块来源数值
- **预览模式** — 无需连接设备即可预览所有模块的强度计算结果
- **配置导入导出** — 所有设置可保存/加载为 JSON 文件

## 连接方式

### 本地模式（推荐）

应用启动内置 WebSocket 服务器并生成二维码，用 DG-LAB App 扫码即可连接。

### 远程模式

连接到远程 WebSocket 服务器地址，支持断线自动重连。

## 安装与运行

**环境要求：** Python 3.10+，Windows（应用监控依赖 Windows API）

```bash
pip install -r requirements.txt
python main.py
```
